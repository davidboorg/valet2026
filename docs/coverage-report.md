## Topp-prioriterade luckor att fylla

Sex specifika val där materialet är historiskt viktigt OCH där realistisk
chans finns att hitta digitaliserat:

1. **Socialdemokraterna 1932** (Per Albins genombrott, folkhemmet) — sök ARAB.
2. **Bondeförbundet 1936** (kohandeln med S) — Centerpartiets eget arkiv.
3. **Moderaterna 1991** (Bildt blir statsminister) — moderaterna.se historik.
4. **Vänsterpartiet 1991** (namnbyte från VPK) — ARAB.
5. **Miljöpartiet 1988** (första riksdagen) — DigitaltMuseum, mp.se historik.
6. **KD 1991** (in i riksdagen genom Alf Svensson) — kdu.se eller KD-arkivet.

## Källor som har material men inte är systematiskt skrapade

- **Riksarkivet** (riksarkivet.se) — Moderata samlingspartiets arkiv I (1904-1965), Centerpartiets arkiv. Kräver beställning eller fysiskt besök.
- **ARAB / Arbetarrörelsens arkiv** (arbark.se) — S, V, SKP-material 1900-1990. ~50 000 affischer i samlingen.
- **Centrum för Näringslivshistoria (CFN)** — Moderaternas, Liberalernas och Centerpartiets historiska arkiv.
- **DigitaltMuseum.se** — Nordiska museet, läns- och stadsmuseer har scattered material. API möjligt.
- **KB:s ej-digitaliserade samling** — kan beställas digitalisering mot betalning (200-500 kr per affisch).
- **Partikanslierna** — egna arkiv. Kräver direktkontakt med S/M/C/V/L/MP/KD/SD-kanslier.

## Rekommendation

Sajten kan gå live nästa vecka med dessa 65 affischer som demonstrationsmaterial. För
fullständig täckning av alla 35 riksdagsval × 8 partier (= 280 möjliga celler, varav
~250 realistiska efter att ha tagit hänsyn till partiernas startår) krävs:

- **Kort sikt (1-2 veckor):** Systematisk scraping av DigitaltMuseum + Stockholmskällan via API. Kan ge ytterligare 30-50 affischer.
- **Medellång sikt (1-3 månader):** Partneravtal med ARAB ger åtkomst till deras digitaliserade S/V-bestånd.
- **Lång sikt:** KB-beställning för 1948-1985-eran (mest kritisk lucka).
- **Etiskt:** För fullständighet behövs tillstånd från partikanslierna för moderna affischer post-1955.
